# Traffic Light System

This JavaScript program checks the color of the traffic light and displays the correct action to follow.

## Concepts Used

* Variables
* if statements
* Conditional Statements

## Code

```javascript
let color = "red";

// Traffic light System
if (color === "red") {
    console.log("Stop! light color is red");
}

if (color === "yellow") {
    console.log("Slow down. light color is yellow");
}

if (color === "green") {
    console.log("Go! light color is green");
}
```

## Output

* Red → Stop
* Yellow → Slow Down
* Green → Go
